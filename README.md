# DNA Sequence Analyzer

A beginner-level bioinformatics project built using Python and Flask.

## Features

- DNA sequence validation
- Nucleotide counting (A, T, G, C)
- GC content calculation
- Reverse complement generation
- DNA to RNA transcription
- Protein translation

## Technologies Used

- Python
- Flask
- HTML
- CSS

## How to Run

1. Install Flask

pip install flask

2. Run application

python app.py

3. Open browser

http://127.0.0.1:5000
